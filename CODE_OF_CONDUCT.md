# 贡献者公约

我们承诺为社区中每一位成员提供友善、安全、欢迎的体验，无论其经验水平、性别、性别认同与表达、性取向、外貌、身体条件、种族、民族、信仰、国籍或政治立场。

## 我们的准则

营造**正向环境**的行为示例：

- 对他人展示友善、尊重与共情
- 欢迎不同的观点与经验
- 接受建设性批评，同时乐于给出有同理心的反馈
- 关注的不仅是个人，更是整个社区的福祉

**不可接受**的行为：

- 使用带有性意味的语言或 imagery，以及任何形式的性骚扰
- 钓鱼、侮辱性评论、人身或政治攻击
- 公开或私下骚扰他人
- 未经明确许可发布他人私人信息（如住址、电子邮箱）
- 其他在专业环境下可被合理认定为不当的行为

## 维护者的职责

项目维护者有权删除、编辑或拒绝不符合本公约的评论、提交、代码、wiki 编辑、issue 和其他贡献。维护者自身也需遵守公约，在代表项目时的言行应体现社区精神。

## 适用范围

本公约适用于所有项目空间（代码仓库、issue、PR、讨论区、文档），以及个人在公共空间代表项目或社区时的行为。

## 执行

辱骂、骚扰及其他不可接受的行为可通过私密渠道联系项目维护者报告（见 [SECURITY.md](SECURITY.md) 中的联系方式）。所有投诉都会被及时、公正地审阅与处理。

本公约改编自 [Contributor Covenant][homepage] 2.1 版。
